# BYOJoystick Extended

BYOJoystick Extended allows you to use your HOTAS, joystick, keyboard, and other input devices with VTOL VR — including modded aircraft not supported by the original mod.

It features a user friendly interface for binding controls on a per-aircraft basis and supports both multiplayer and multicrew vehicles. The mod is **client-side only** — other players on a multiplayer server do not need BYOJoystick installed.

## Supported Aircraft

### Base Game
- AV-42C
- AH-94 (Front & Rear)
- AH-6
- A-10D Warthog
- F/A-26B
- F-45A
- F-16C Viper *(completed)*
- F-22A Raptor
- T-55 (Front & Rear)
- EF-24G (Front & Rear)

### Modded Aircraft *(new in this fork)*
- **F-16C Viper** — Full MFD, systems, lights, and radio support
- **SU-47M Berkut** — Full support including Wing Fold, dual engines, TVC, and CMFD
- **F-5E Tiger III** — Full support including VRButton-based MFDs, dual engines, and carrier ops controls
- **AV-8B Harrier** — Full support including nozzle tilt, hover AP, water injection, refuel probe, and dual MFD edge keys
